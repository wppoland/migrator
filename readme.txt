== Description ==

Back up your WordPress site and move it to a new host. One file, drag and drop, no technical setup.

== Installation ==

1. Upload the plugin to `/wp-content/plugins/migrator`, or install via Plugins → Add New.
2. Activate it. WooCommerce must be active.
